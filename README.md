# Built With:
- HTML5
- CSS3
- Jquery
- Foundation

# Install

This was setup using foundation CLI 6

- Clone Repo
- Install Depends.
  - run `npm install`
  - run `bower install`
- Run Browser-Sync
  - `foundation watch`
